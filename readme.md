2024.06.03 HTML 설계 정리
-index(H) : 홈페이지
  - menu(header) : 아이콘(베리향기) / 정보+상품정보+문의사항
  - 오시는 길
  - 농장사진(background) : 농장 소개글
  - 블루베리 소개 글(타임즈 기사/슈퍼푸드)
- 정보(H) : 블루베리 정보
  - 블루베리 소개(타임즈 기사/슈퍼푸드)
  - 블루베리 효능(건강정보-비타민)
  - 수확 기간
  - 블루베리의 종류 (종류별 특징)
- 상품 정보(H) : 상품의 규격
  - 사이즈 정보
  - 가격 정보
  - 용량 정보
  - 종류 별 구매
  - 소비기한
- 문의사항
  - 전화번호(판매기간/가능시간)
  - 카카오톡
  - 사이트 문의
